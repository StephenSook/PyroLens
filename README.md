# PyroLens

**PyroLens** is a full-stack prescribed burn planning and monitoring platform for forestry and land management teams. It combines ESP32 field sensors, weather and satellite data (e.g. NDVI), and ML-based burn-window scoring so planners can see when and where conditions are safer for controlled burns. A React + Vite dashboard surfaces recommendations, maps, and impact-style metrics; a FastAPI backend orchestrates APIs, data ingestion, and PostgreSQL (including Supabase) storage.
